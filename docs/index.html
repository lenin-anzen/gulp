Task Runner - Gulp

Permite la gestión y automatización de tareas mediante streams(flujos de datos).

Para realizar operaciones con el stream se utilizan pipes.

Instalación

Instalar globalmente el CLI(Command Line Interface) de Gulp:

npm install -g gulp-cli

Instalar Gulp en las dependencias de desarrollo del proyecto

mkdir gulp-project && cd gulp-project
npm init --yes
npm install --save-dev gulp

Generar archivo gulpfile.js con la tarea predeterminada:

'use strict';

const gulp = require('gulp');

gulp.task('default', function() {
  console.log('Gulp instalado correctamente!');
});

Ejecutar tarea predeterminada:

gulp
[12:09:06] Using gulpfile ~\gulp-project\gulpfile.js
[12:09:06] Starting 'default'...
Gulp instalado correctamente!
[12:09:06] Finished 'default' after 898 μs
gulp.task(name [, deps] [, fn])

Define una tarea con los siguientes parámetros:

  1. name (string): el nombre de la tarea

  2. deps (array): dependencias de otras tareas

  3. fn (function): la función a ejecutar

Puede regresar un stream o una promesa.

gulp.src(globs[, options])

Obtiene una cadena o un array que coincida con los globs(archivos) a utilizar:

  1. globs (string o array): el nombre de los archivos.

  2. options (object): opciones adicionales.

Regresa un pipe que puede ser usado por un plugin o hacia gulp.dest.

gulp.dest(path[, options])

Obtiene un stream y escribe archivos hacia un directorio de destino:

  1. path (string o function): el nombre del directorio donde se escribirán los archivos.

  2. options (object): opciones adicionales.

gulp.watch(glob [, opts], tasks)

Observa cambios en los archivos y realiza una acción cuando dichos archivos son modificados:

  1. glob (string o array): el nombre de los archivos.

  2. options (object): opciones adicionales.

  3. tasks (array): Nombres de las tareas a ejecutar.

Regresa un EventEmitter que emite eventos change.

Plugins

Existen cerca de 3206 plugins, una muestra de su uso es:.

gulp-util

Funciones para los plugins de gulp.

npm i --save-dev gulp-util

Ejemplo:

gutil.log(gutil.colors.magenta('Cambio de color'));
gulp-concat

Concatena archivos

npm i --save-dev gulp-concat

Ejemplo:

gulp.task('concatenate', function() {
  return gulp.src('./src/lib/*.js')
    .pipe(concat('all.js'))
    .pipe(gulp.dest('./dist/'));
});
gulp-uglify

Minificación de archivos:

npm i --save-dev gulp-uglify

Ejemplo:

gulp.task('minify', function() {
  return gulp.src('./src/lib/*.js')
    .pipe(concat('min.js'))
    .pipe(uglify())
    .pipe(gulp.dest('./dist/'));
});
gulp-eslint

Permite ejecutar ESLint mediante Gulp

npm i --save-dev gulp-eslint eslint

Ejemplo:

gulp.task('lint', function() {
  return gulp.src(['**/*.js','!node_modules/**'])
  	.pipe(eslint())
    .pipe(eslint.format())
  	.pipe(eslint.failAfterError());
});
gulp-sourcemaps

Generar source maps:

npm i --save-dev gulp-sourcemaps

Ejemplo:

gulp.task('scrmap', function() {
  gulp.src('./src/**/*.js')
    .pipe(sourcemaps.init())
    .pipe(concat('min-srcmap.js'))
    .pipe(uglify())
    .pipe(sourcemaps.write())
    .pipe(gulp.dest('dist'));
});
gulp-hub

Permite incluir tareas de diferentes archivos:

npm i --save-dev gulp-hub

Ejemplo:

hub("./src/gulp_tasks/*.js");
gulp-sequence

Permite ejecutar tareas en paralelo:

npm i --save-dev gulp-sequence

Ejemplo:

gulp.task('sequence', gulpSequence(
    ['lint', 'concatenate'],
    ['minify', 'srcmap']
  ));
